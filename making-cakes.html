How to add your clips

Each placeholder has a comment in the code showing exactly what to paste in. In short: put your video file (e.g. lesson1.mp4) in the same folder as this page, then replace the <div class="placeholder">...</div> with a <video controls> tag pointing at it.

Keep each clip compressed for the web (around 720p is plenty) so the page stays fast to load. A poster image is optional but gives viewers a nice still frame before they hit play.
